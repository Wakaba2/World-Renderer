# World Renderer  
This plugin allows you to render objects on the sky and world  
  
![gif](https://i.imgur.com/H9lEztq.png)  
  
# Changelog

Added a procedure "Set color mask"  
Fixed some bugs

Fixed some bugs

v1.2.1  
Fixed some bugs  
  
v1.2.0
